## List Your Recordings
The Plaud API makes it easy to fetch a complete list of your audio recordings. Use the following example to quickly retrieve all your recordings:

<CodeGroup dropdown>

```bash bash
curl -H "Authorization: <YOUR_API_KEY>" https://platform.plaud.ai/api/files/
```

```python python
import requests

headers = {'Authorization': '<YOUR_API_KEY>'}
resp = requests.get('https://platform.plaud.ai/api/files/', headers=headers)
print(resp.json())
```
</CodeGroup>

## Retrieve Recording Details
You can retrieve the detail of an audio recording by specifying `file_id`:

<CodeGroup dropdown>

```bash bash
curl -H "Authorization: <YOUR_API_KEY>" https://platform.plaud.ai/api/files/{file_id}
```

```python python
import requests

file_id = "your_file_id"  # Replace with actual file ID
headers = {'Authorization': '<YOUR_API_KEY>'}
resp = requests.get(f'https://platform.plaud.ai/api/files/{file_id}', headers=headers)
print(resp.json())
```
</CodeGroup>
