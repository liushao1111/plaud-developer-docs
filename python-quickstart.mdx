# Quickstart Guide

## 1. Get Your API Key

Sign up and generate your API key from the [Plaud Developer Console](https://console.plaud.ai).

## 2. List Your Recordings

To list your audio files (recordings):

```bash
curl -H "Authorization: <YOUR_API_KEY>" https://api.plaud.ai/files/
```

**Python Example:**

```python
import requests

headers = {'Authorization': '<YOUR_API_KEY>'}
resp = requests.get('https://api.plaud.ai/files/', headers=headers)
print(resp.json())
```

## 3. Retrieve Recording Details

To get details for a specific recording:

```bash
curl -H "Authorization: <YOUR_API_KEY>" https://api.plaud.ai/files/{file_id}
```

## 4. Explore More Endpoints

The Plaud API offers many more capabilities:

| Resource | Endpoint | Description |
|----------|----------|-------------|
| Devices | `GET /devices/` | List all your devices |
| Users | `GET /users/` | List all users |
| Tenants | `GET /tenants/` | List all tenants |
| Transcriptions | File details | Check `ai_status` and `ai_result_list` fields |

See the [API Reference](/api-reference) for complete documentation of all endpoints.
