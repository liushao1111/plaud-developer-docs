The Plaud API provides a simple interface to state-of-the-art recording devices and features. 
Follow this guide to learn how to bind your Plaud device. 
See the [developer guides](/documentation/get_started/quickstart#explore-our-developer-guides) for more examples with our other products.

<Tip>
**Note:**  
This documentation site is for SaaS companies that want to integrate directly with Plaud devices and leverage Plaud’s ASR, diarization, and AI pipelines in their own SaaS product. If you are interested in integrating with existing Plaud users' accounts via the OAuth API (currently in early beta), please fill out [this survey here](https://docs.google.com/forms/d/e/1FAIpQLSdndvVU1mcRGg_E7YURPEtB13R1zHs55T-234AC6STJyW1n8w/viewform) to join the waitlist and we will notify you as soon as it’s available..
</Tip>


## Prerequisites
1. You need to have an iOS/Android app that your users can download, OR
2. Your users need to have bluetooth-enabled computers/laptops that they can use to transfer recordings over to your web app

## Using the API

<Steps>
  <Step title="Create an APP">
    [Create an APP in the dashboard here,]() which you'll use to securely [access the API](/api_guide/api_intro/authorization).
    <Tip>
      Don’t have access yet? Book a call with our [Submit Request Form](https://docs.google.com/forms/d/e/1FAIpQLSeMr2cRAjrWK79k6HuPOjeOoU7bqHDbk1ZQTsH3RZ8rhJPwAg/viewform) here.
    </Tip>
    Store `Client ID` and `Secret Key` as managed secrets as a environment variable via an `.env` file, or directly in your app’s configuration depending on your preference.
    ```bash .env
    PLAUD_CLIENT_ID=<PLAUD_CLIENT_ID>
    PLAUD_CLIENT_SECRET_KEY=<PLAUD_CLIENT_SECRET_KEY>
    ```
  </Step>
  <Step title="Install Dependencies">
    We’ll also use the `dotenv` library to load our Client ID and Secret Key from environment variables.
    <CodeGroup>
    ```python Python
    pip install python-dotenv
    ```
    </CodeGroup>
  </Step>
  <Step title="Make your first request">
    Create a new file named `example.py`, depending on your language of choice and add the following code:
    <CodeGroup>
    ```python Python
    import base64
    import requests
    from dotenv import load_dotenv

    load_dotenv()

    # Step 1: Generate API token
    client_id = os.getenv("PLAUD_CLIENT_ID")
    secret_key = os.getenv("PLAUD_CLIENT_SECRET_KEY")
    
    # Create base64 encoded credentials
    credentials = base64.b64encode(f"{client_id}:{secret_key}".encode()).decode()
    
    # Request API token
    token_response = requests.post(
        'https://platform.plaud.ai/api/oauth/api-token',
        headers={
            'Authorization': f'Bearer {credentials}',
            'Content-Type': 'application/json'
        }
    )
    
    token_data = token_response.json()
    api_token = token_data['api_token']
    
    # Step 2: Use token to get device list
    devices_response = requests.get(
        'https://platform.plaud.ai/api/devices/',
        headers={
            'Authorization': f'Bearer {api_token}',
            'Content-Type': 'application/json'
        }   
    )

    devices = devices_response.json()
    print(devices)
    ```
    </CodeGroup>
  </Step>
  <Step title="Run the code">
    <CodeGroup>
    ```python Python
    python example.py
    ```
    </CodeGroup>
    You can get a device list managed under your app.
  </Step>
</Steps>
