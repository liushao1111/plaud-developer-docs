This guide provides a reference for all webhook events under whole Plaud lifecycle. 
For a tutorial on handling webhooks, see the [File Webhooks guide](/documentation/developer_guides/tutorials/files/webhooks).

## Transcription Events

| Event Type | Description | Payload |
|:-----------|:------------|:--------|
| **audio_transcribe.completed** | Transcription processing finished | Task info, file ID, transcript results |
| **audio_transcribe.failed** | Transcription processing failed | Task info, file ID, error details |
| **audio_transcribe.started** | Transcription processing started | Task info, file ID, estimated completion time |

## Device Events

| Event Type | Description | Payload |
|:-----------|:------------|:--------|
| **device.created** | New device registered | Device information |
| **device.bound** | Device bound to user | Device and user information |
| **device.unbound** | Device unbound from user | Device and previous user information |
| **device.updated** | Device information updated | Updated device information |
